# Educational Products App

Aplicação mobile desenvolvida em Flutter com persistência local em SQLite para gerenciamento de produtos e materiais educacionais.

## Funcionalidade

CRUD completo de produtos educacionais: cadastrar, listar, editar e remover itens, com dados salvos localmente no dispositivo.

## Tecnologias

- Flutter
- Dart
- SQLite (`sqflite`)
- `path`

## Estrutura

```
lib/
├── database/
├── models/
├── repositories/
├── controllers/
└── pages/
```

## Como executar

```bash
git clone URL_DO_REPOSITORIO
cd educational_products_app
flutter pub get
flutter run
```

> O projeto utiliza `sqflite` e deve ser executado em Android ou iOS.
